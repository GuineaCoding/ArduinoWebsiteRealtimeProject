  • Android Studio

Setting Up Your Firebase Database

To get started, you'll need to create a Firebase database. Here's what you need to know:

  • Head over to the Firebase website and sign up or log in.
  • Once you're in the Firebase console, create a new project.
  • During the setup process, it's important to specifically choose the 'Web App' option to ensure compatibility with the project.
  • Follow the on-screen instructions to complete the setup.

Installation Process of the Ionic Project

Let's proceed with the installation process of the Ionic project. I recommend watching the video first:

Clone and Install the Project:

  1. Clone the project from GitHub to your local machine.
  2. Open a terminal or command prompt in the project directory.
  3. Run npm install to install all required dependencies.
  4. Create the environments folder in the src folder.
  5. In the environments folder, create the environment.ts file and place the Firebase keys there.

Run the Ionic App:

Within the project directory, run ionic serve to start the app in development mode. Open your web browser and navigate to http://localhost:8100 to view the app.

Important: Due to a misconfiguration issue between the latest version of AngularFire and Ionic, an error may occur. This can be fixed by following the advice in this GitHub issue.

Building the Ionic App:

For Android APK, ensure you have Android Studio installed and then follow Ionic's instructions from article to create the android folder Link here Run ionic build to build the app for production.

Note: Ensure that your Firebase project settings and configuration are correctly integrated into the Ionic app. This includes updating the Firebase configuration in the appropriate files within the Ionic project.

Setting Up the Arduino Project

To get started with the Arduino code in the IDE, you'll first need to clone the project repository. Here's how:

  1. Make a Git clone of the project from GitHub.
  2. After cloning, you'll need to create a config.h file in your project directory.
  3. In the config.h file, add your WiFi name, WiFi password, Firebase host, and Firebase secret.
